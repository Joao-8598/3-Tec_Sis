EX003 - Cálculo de Média

Este programa calcula a média aritmética de três notas fornecidas pelo usuário.

Saída:

Dicas:

  • A função input() captura dados do usuário como string
  • float() converte uma string para número decimal
  • Variáveis (n1, n2, n3, m) armazenam valores
  • Operadores aritméticos (+, /) realizam cálculos
  • Parênteses definem a ordem de precedência nas operações
  • f-strings permitem formatação (ex: :.1f para 1 casa decimal)
  • O cálculo da média é feito pela soma das notas dividida por 3
